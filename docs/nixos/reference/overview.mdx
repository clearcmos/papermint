### 📋 Command Quick References

**NixOS System Management**:
```bash
# System rebuilds
nixos-rebuild switch --flake .#$(hostname)
nixos-rebuild test --flake .#$(hostname)
nixos-rebuild dry-run --flake .#$(hostname)

# Package management
nix search nixpkgs <package-name>
nix-env -iA nixos.<package-name>
nix-collect-garbage -d

# Service management
systemctl status <service>
systemctl start|stop|restart <service>
journalctl -u <service> -f
```

**Infrastructure Commands**:
```bash
# Remote deployment
nixos-rebuild switch --flake .#misc --target-host misc.home.arpa

# Secret management (from /etc/nixos/secrets)
nix run github:ryantm/agenix -- -e secret.age -i ../keys/host-key

# AI services
curl -s http://localhost:11434/api/tags | jq
systemctl status ollama

# Network diagnostics
ping -c 4 hostname
dig hostname
nmap -p 22,80,443 hostname
```

### 🌐 Network Architecture Reference

**IP Address Allocations**:
```
Network: 192.168.1.0/24
├── 192.168.1.1    - Router/Gateway
├── 192.168.1.2    - cmos.home.arpa (desktop)
├── 192.168.1.5    - misc.home.arpa (server)
├── 192.168.1.10   - jimmich.home.arpa (server)
└── 192.168.1.14   - nixvm.home.arpa (VM)
```

**Port Allocations**:
```
Core Services:
├── 22/tcp    - SSH (all hosts)
├── 80/tcp    - HTTP (servers)
├── 443/tcp   - HTTPS (servers)
├── 9090/tcp  - Cockpit (all hosts)
├── 445/tcp   - SMB/CIFS (all hosts)
└── 5353/udp  - mDNS (all hosts)

Host-Specific:
├── cmos:
│   ├── 8001/tcp  - radeontop-web
│   └── 11434/tcp - Ollama
└── misc:
    ├── 8080/tcp  - Various web services
    └── 8090/tcp  - Additional services
```

### 🔐 Security Configuration Reference

**Authentication Flow**:
```
External Request → Cloudflare → Traefik → OAuth2-Proxy → Service
                                              ↓
                                          Keycloak (OIDC)
                                              ↓
                                     Google SSO + User DB
```

**TLS Configuration**:
- **Minimum Version**: TLS 1.2
- **Cipher Suites**: Modern, secure ciphers only
- **Certificate Authority**: Let's Encrypt
- **HSTS**: Enabled with preload
- **Certificate Renewal**: Automated

## 📝 Glossary

### 🎯 Infrastructure Terms

**A**
- **Agenix**: Age-based secret encryption for NixOS
- **API**: Application Programming Interface

**F**
- **Flake**: Nix's new system for reproducible builds
- **FQDN**: Fully Qualified Domain Name

**N**
- **NixOS**: Linux distribution built on the Nix package manager
- **Nix**: Purely functional package manager

**O**
- **OAuth2**: Open standard for access delegation
- **Ollama**: Platform for running large language models locally

**T**
- **Traefik**: Modern reverse proxy and load balancer
- **TLS**: Transport Layer Security

### 🔒 Security Terms

**2FA**: Two-Factor Authentication  
**CSRF**: Cross-Site Request Forgery  
**HSTS**: HTTP Strict Transport Security  
**JWT**: JSON Web Token  
**OIDC**: OpenID Connect  
**PKCE**: Proof Key for Code Exchange  
**SSO**: Single Sign-On  
**WAF**: Web Application Firewall

### 🛠️ Technical Terms

**API**: Application Programming Interface  
**CDN**: Content Delivery Network  
**CORS**: Cross-Origin Resource Sharing  
**DNS**: Domain Name System  
**HTTP/HTTPS**: HyperText Transfer Protocol (Secure)  
**JSON**: JavaScript Object Notation  
**REST**: Representational State Transfer  
**SLA**: Service Level Agreement  
**URL**: Uniform Resource Locator

## 🛠️ Quick Reference Cards

### 🚀 Emergency Commands

**System Recovery**:
```bash
# Boot previous generation
# (Select from GRUB menu)

# Emergency rollback
nixos-rebuild switch --rollback

# Service emergency restart
systemctl restart <critical-service>

# Check system health
systemctl --failed
journalctl --priority=err --since="1 hour ago"
```

**Network Diagnostics**:
```bash
# Basic connectivity
ping -c 4 8.8.8.8

# DNS resolution
nslookup hostname
dig hostname

# Service availability
curl -I https://service.domain.com
telnet hostname port
```

### 📊 Monitoring Commands

**System Status**:
```bash
# Overall system health
systemctl status
df -h
free -h
uptime

# Service-specific status
systemctl status ollama
systemctl status keycloak
systemctl status traefik
systemctl status fail2ban
```

**Performance Monitoring**:
```bash
# Real-time monitoring
htop
iotop
nethogs
watch "df -h"

# Historical analysis
journalctl --since="1 hour ago"
sar -u 1 10
iostat -x 1 10
```
