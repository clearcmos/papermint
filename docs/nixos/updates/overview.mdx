#### Known Issues & Resolutions
1. **Obsidian build failure** (cmos):
   - Issue: IPv6 connectivity to GitHub releases during fetch
   - Resolution: Prefetched tarball + added `--ipv4` curl option
   - Workaround used: `nix-prefetch-url` before rebuild

2. **oauth2-proxy startup race condition** (misc):
   - Issue: Service started before Keycloak/Traefik were ready
   - Resolution: Added systemd service dependencies
   - Impact: Glances authentication now works

#### Rollback Instructions
**To rollback cmos to old packages:**
1. Edit `flake.nix` line 6, uncomment the rollback line
2. Rebuild: `nixos-rebuild boot --flake .#cmos`
3. Reboot

**To rollback misc:**
```bash
git checkout HEAD~2 -- flake.lock  # Revert to 7c815e5
nixos-rebuild switch --flake .#misc --target-host misc.home.arpa
```

**To rollback documentation:**
```bash
git revert 71c388d
nixos-rebuild switch --flake .#misc --target-host misc.home.arpa
```
