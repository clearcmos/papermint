## Performance Overview

The system is optimized for dual workloads: high-performance AI processing and AAA gaming (WoW The War Within). This configuration provides better AI performance than the previous setup while reserving resources for gaming.

### Performance Metrics

**Current vs Previous Setup**:
- ✅ **50% more concurrency**: 6 parallel requests (was 4)
- ✅ **2x larger queue**: 1024 requests (was 512)
- ✅ **Same context quality**: 8K tokens maintained
- ✅ **Better resource efficiency**: Dynamic loading vs static allocation
- ✅ **Gaming compatibility**: 6-7GB VRAM reserved for WoW

## System Specifications

### Hardware Configuration
- **GPU**: AMD Radeon RX 6800 XT (16GB VRAM)
- **CPU**: Multi-core with 8 threads allocated to AI
- **RAM**: 32GB+ (system memory for model loading)
- **Storage**: NVMe SSD for fast model loading

### Software Stack
- **OS**: NixOS with ROCm support
- **GPU Driver**: ROCm with gfx1030 optimization
- **AI Engine**: Ollama with ROCm acceleration
- **Quantization**: Q4_K_M for optimal balance

### Benchmarking Tools

**Response Time Benchmarking**:
```bash
#!/bin/bash
# Benchmark script for AI response times

models=("qwen2.5-coder:7b-instruct" "qwen2.5:7b-instruct")
prompt="Write a hello world program in Python"

for model in "${models[@]}"; do
  echo "Testing $model..."
  time curl -X POST http://localhost:11434/api/generate \
    -H "Content-Type: application/json" \
    -d "{\"model\": \"$model\", \"prompt\": \"$prompt\", \"stream\": false}" \
    | jq -r '.response' | wc -w
done
```

**Concurrent Load Testing**:
```bash
#!/bin/bash
# Load testing with multiple concurrent requests

concurrent_requests=6
model="qwen2.5:7b-instruct"

echo "Testing $concurrent_requests concurrent requests..."
start_time=$(date +%s.%N)

for i in $(seq 1 $concurrent_requests); do
  curl -X POST http://localhost:11434/api/generate \
    -d "{\"model\": \"$model\", \"prompt\": \"Test request $i\"}" \
    > /dev/null 2>&1 &
done

wait
end_time=$(date +%s.%N)
total_time=$(echo "$end_time - $start_time" | bc)

echo "Total time for $concurrent_requests requests: ${total_time}s"
echo "Average time per request: $(echo "$total_time / $concurrent_requests" | bc -l)s"
```

## Troubleshooting Performance

### Common Performance Issues

**Slow Response Times**:
1. Check GPU memory usage: `rocm-smi --showmeminfo vram`
2. Monitor CPU usage: `htop`
3. Check model loading: `curl -s http://localhost:11434/api/ps`
4. Verify disk I/O: `iotop`

**High VRAM Usage**:
1. Check loaded models: `curl -s http://localhost:11434/api/ps`
2. Force model unloading: `curl -X POST ... -d '{"keep_alive": 0}'`
3. Restart service: `systemctl restart ollama.service`

**Gaming Performance Drops**:
1. Check AI model usage: `rocm-smi --showmeminfo vram`
2. Verify model limits: `systemctl show ollama.service | grep OLLAMA_MAX`
3. Monitor concurrent requests: `ss -tuln | grep 11434`
