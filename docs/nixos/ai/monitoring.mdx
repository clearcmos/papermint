## Monitoring Overview

The AI system includes multiple monitoring layers to ensure optimal performance, resource utilization, and early detection of issues. This guide covers system health checks, performance monitoring, and troubleshooting tools.

### Gaming Performance Impact

**VRAM Allocation Monitoring**:
```bash
#!/bin/bash
# Monitor VRAM allocation for gaming compatibility

total_vram=$(rocm-smi --showmeminfo vram --csv | grep -E "^[0-9]" | cut -d',' -f3 | sed 's/ MB//')
used_vram=$(rocm-smi --showmeminfo vram --csv | grep -E "^[0-9]" | cut -d',' -f2 | sed 's/ MB//')
available_vram=$((total_vram - used_vram))

echo "VRAM Status for Gaming:"
echo "  Total: ${total_vram}MB"
echo "  Used by AI: ${used_vram}MB" 
echo "  Available for WoW: ${available_vram}MB"

if [ $available_vram -gt 6000 ]; then
  echo "  Status: ✅ Sufficient VRAM for gaming"
elif [ $available_vram -gt 4000 ]; then
  echo "  Status: ⚠️  Limited VRAM for gaming"
else
  echo "  Status: ❌ Insufficient VRAM for gaming"
fi
```

## Model Performance Monitoring

### Model Loading and Response Times

**Response Time Monitoring**:
```bash
#!/bin/bash
# Benchmark model response times

models=("qwen2.5-coder:7b-instruct" "qwen2.5:7b-instruct" "bge-m3")

for model in "${models[@]}"; do
  echo "Testing $model..."
  
  # Cold start (model not loaded)
  curl -X POST http://localhost:11434/api/generate \
    -d "{\"model\": \"$model\", \"keep_alive\": 0}" > /dev/null 2>&1
  
  start_time=$(date +%s.%N)
  response=$(curl -s -X POST http://localhost:11434/api/generate \
    -d "{\"model\": \"$model\", \"prompt\": \"test\", \"stream\": false}")
  end_time=$(date +%s.%N)
  
  response_time=$(echo "$end_time - $start_time" | bc -l)
  
  if [[ $model == "bge-m3" ]]; then
    # Test embeddings for bge-m3
    start_time=$(date +%s.%N)
    curl -s -X POST http://localhost:11434/api/embeddings \
      -d "{\"model\": \"$model\", \"prompt\": \"test\"}" > /dev/null
    end_time=$(date +%s.%N)
    response_time=$(echo "$end_time - $start_time" | bc -l)
  fi
  
  printf "  Response time: %.2fs\n" $response_time
  echo
done
```

### Concurrent Request Monitoring

**Load Testing**:
```bash
#!/bin/bash
# Test concurrent request handling

model="qwen2.5:7b-instruct"
concurrent_requests=6
prompt="Test concurrent request"

echo "Testing $concurrent_requests concurrent requests..."
start_time=$(date +%s.%N)

for i in $(seq 1 $concurrent_requests); do
  curl -s -X POST http://localhost:11434/api/generate \
    -d "{\"model\": \"$model\", \"prompt\": \"$prompt $i\", \"stream\": false}" \
    > "/tmp/response_$i.json" &
done

wait
end_time=$(date +%s.%N)

total_time=$(echo "$end_time - $start_time" | bc -l)
avg_time=$(echo "$total_time / $concurrent_requests" | bc -l)

printf "Total time: %.2fs\n" $total_time
printf "Average time per request: %.2fs\n" $avg_time

# Cleanup
rm -f /tmp/response_*.json
```

## Performance Alerting

### Automated Monitoring Scripts

**Health Check Script** (`/etc/ollama-health.sh`):
```bash
#!/bin/bash
# Comprehensive health check with alerting

VRAM_THRESHOLD=90    # Alert if VRAM usage > 90%
RESPONSE_THRESHOLD=5 # Alert if response time > 5 seconds
TEMP_THRESHOLD=80    # Alert if GPU temp > 80°C

# Check service status
if ! systemctl is-active ollama.service > /dev/null; then
  echo "ALERT: Ollama service is not running"
  exit 1
fi

# Check VRAM usage
vram_used=$(rocm-smi --showmeminfo vram --csv | grep -E "^[0-9]" | cut -d',' -f2 | sed 's/ MB//')
vram_total=$(rocm-smi --showmeminfo vram --csv | grep -E "^[0-9]" | cut -d',' -f3 | sed 's/ MB//')
vram_percent=$((vram_used * 100 / vram_total))

if [ $vram_percent -gt $VRAM_THRESHOLD ]; then
  echo "ALERT: High VRAM usage: ${vram_percent}%"
fi

# Check API response time
start_time=$(date +%s.%N)
curl -s http://localhost:11434/api/version > /dev/null
end_time=$(date +%s.%N)
response_time=$(echo "$end_time - $start_time" | bc -l)

if (( $(echo "$response_time > $RESPONSE_THRESHOLD" | bc -l) )); then
  printf "ALERT: Slow API response: %.2fs\n" $response_time
fi

# Check GPU temperature
gpu_temp=$(rocm-smi --showtemp --csv | grep -E "^[0-9]" | cut -d',' -f2 | sed 's/°C//')
if [ $gpu_temp -gt $TEMP_THRESHOLD ]; then
  echo "ALERT: High GPU temperature: ${gpu_temp}°C"
fi

echo "Health check completed: $(date)"
```

**Automated Monitoring Cron Job**:
```bash
# Add to crontab (crontab -e)

*/5 * * * * /bin/bash /etc/ollama-health.sh >> /var/log/ollama-health.log 2>&1

# Daily performance report
0 6 * * * /bin/bash /etc/ollama-models.sh test >> /var/log/ollama-performance.log 2>&1
```

**Ollama Service Logs**:

### Performance Logs

**Custom Performance Logging**:

## Monitoring Dashboard Setup

### Simple Dashboard Script

**Real-Time Dashboard** (`/usr/local/bin/ollama-dashboard`):
```bash
#!/bin/bash
# Simple monitoring dashboard

while true; do
  clear
  echo "╔════════════════════════════════════════════════════════════════════╗"
  echo "║                    OLLAMA AI SYSTEM DASHBOARD                     ║"
  echo "╚════════════════════════════════════════════════════════════════════╝"
  echo
  
  # Service Status
  echo "🔧 SERVICE STATUS:"
  echo "   Ollama: $(systemctl is-active ollama.service)"
  echo "   API: $(curl -s -w 'Response: %{http_code} (%{time_total}s)' http://localhost:11434/api/version -o /dev/null)"
  echo
  
  # Models
  echo "🧠 MODELS:"
  installed_count=$(curl -s http://localhost:11434/api/tags | jq -r '.models | length')
  loaded_count=$(curl -s http://localhost:11434/api/ps | jq -r '.models | length')
  echo "   Installed: $installed_count"
  echo "   Loaded: $loaded_count"
  echo
  
  # GPU Status
  echo "🎮 GPU STATUS:"
  vram_info=$(rocm-smi --showmeminfo vram --csv | grep -E "^[0-9]" | cut -d',' -f2,3)
  echo "   VRAM: $vram_info"
  
  gpu_temp=$(rocm-smi --showtemp --csv | grep -E "^[0-9]" | cut -d',' -f2)
  echo "   Temperature: $gpu_temp"
  
  gpu_util=$(rocm-smi --showuse --csv | grep -E "^[0-9]" | cut -d',' -f2)
  echo "   Utilization: $gpu_util"
  echo
  
  # System Load
  echo "💻 SYSTEM LOAD:"
  echo "   $(uptime)"
  echo
  
  echo "Press Ctrl+C to exit | Refreshing in 5 seconds..."
  sleep 5
done
```

Make it executable:
```bash
sudo chmod +x /usr/local/bin/ollama-dashboard
```

Usage:
```bash
ollama-dashboard
```
