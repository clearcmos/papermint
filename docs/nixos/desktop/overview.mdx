## 🖥️ VM Desktop Environment (nixvm)

### 🌊 labwc Configuration

**nixvm** runs a lightweight labwc desktop optimized for VM usage:

| Component | Description |
|-----------|-------------|
| **Window Manager** | labwc Wayland compositor |
| **Status Bar** | Waybar with aesthetic Catppuccin theme |
| **Launcher** | Wofi application launcher |
| **Notifications** | Mako notification daemon |
| **Window Switcher** | Hyprshell (Alt+Tab) |

**Key Features**:
- **Taskbar**: wlr/taskbar showing running apps with click-to-switch
- **Workspaces**: 5 persistent workspaces (Super+1-5)
- **Floating bar**: Rounded corners with Catppuccin Mocha styling
- **3D Acceleration**: VirtIO-GL enabled for smooth graphics

**Hyprshell Configuration**:
- Supports `HYPRSHELL_SKIP_KEYBINDS=1` to use manual keybind registration
- Alt+Tab window switcher with visual preview
- Configured via `/etc/xdg/hyprshell/config.ron`

**Key Shortcuts**:
| Shortcut | Action |
|----------|--------|
| `Super+Return` | Open terminal (Kitty) |
| `Super+D` / `Super+Space` | Application launcher |
| `Super+Q` | Close window |
| `Super+F` | Toggle fullscreen |
| `Alt+Tab` | Window switcher |
| `Super+1-5` | Switch workspace |

## 🚨 Common Issues & Solutions

### 🖥️ Display Issues

**Multiple Monitor Problems**:
- **Configuration Reset**: **[Declarative multi-monitor setup](kde-multimonitor.md)** (persists across reboots)
- **Scaling Issues**: Per-monitor scaling adjustment
- **Performance**: Compositor optimization for gaming

**Resolution Problems**:
- **Custom Resolutions**: xrandr configuration
- **Refresh Rates**: Optimal gaming vs productivity modes
- **Color Accuracy**: ICC profile management

### 🧊 Compositor Freezes

**Complete Desktop Freeze (audio still works)**:
- **Cause**: Qt apps forcing `QT_QPA_PLATFORM=xcb` under Wayland
- **Symptoms**: XCB SHAPE errors in KWin logs, display frozen, SSH still works
- **Recovery**: Kill offending app via SSH, switch VT with `sudo chvt 2`
- **Full Guide**: **[KWin XCB Freeze](kwin-xcb-freeze.md)**

### 🎮 Gaming Issues

**Performance Problems**:
- **GPU Conflicts**: AI vs gaming scheduling
- **Audio Latency**: PipeWire optimization
- **Input Lag**: Compositor bypass for games

**Compatibility Issues**:
- **Proton**: Steam compatibility layer fixes
- **Anti-cheat**: BattlEye and EAC support
- **VR Gaming**: OpenXR configuration

### 🔊 Audio Issues

**PipeWire Problems**:
- **No Audio**: Service restart procedures
- **Crackling**: Buffer size optimization
- **Multiple Outputs**: Audio routing configuration
