## 🏗️ Host Roles

### 🖥️ Desktop Workstation (cmos)
**Primary Use**: Development, AI/ML workloads, gaming

**Key Features**:
- AMD Radeon RX 6800 XT with ROCm for AI acceleration
- 32GB RAM + 16GB swap for large AI models
- KDE Plasma 6 desktop environment
- Ollama for local LLM inference
- Gaming stack (Steam, Wine, Lutris)

**Services**:
- Ollama AI server (port 11434)
- KDE Remote Desktop (port 8002)
- GPU monitoring (radeontop-web)
- Claude Code WebUI (port 8084)

---

### 🌐 Primary Server (misc)
**Primary Use**: Web services, authentication, reverse proxy

**Key Features**:
- Intel server platform (Mac Mini)
- Central authentication and service hosting
- Public-facing services via Traefik
- PostgreSQL database for multiple services

**Services**:
- Traefik reverse proxy (80/443)
- Keycloak identity provider + OAuth2-proxy
- SABnzbd/Radarr/Sonarr media automation
- Cockpit system management
- Clear NDR (Suricata IDS)
- Security Dashboard
- NixDocs documentation site
- PrivateBin encrypted pastebin
- Memos note-taking

---

### 💾 Secondary Server (jimmich)
**Primary Use**: File services, backup, storage

**Key Features**:
- High storage capacity
- File sharing via Samba
- Backup target for other hosts
- Minimal services (server-only)

**Services**:
- Samba file shares
- SSH access
- Cockpit dashboard
