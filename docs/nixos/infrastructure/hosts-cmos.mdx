## 🎯 Overview

**cmos** is the primary desktop workstation in the infrastructure, featuring a powerful AMD GPU for AI/ML acceleration, extensive development tools, and a complete gaming stack. This host serves as the control center for infrastructure management and handles computationally intensive workloads.

**Role**: Desktop Workstation, AI/ML Platform, Gaming Station, Development Environment

---

## 💻 Hardware Specifications

### System Information
- **Hostname**: `cmos.home.arpa`
- **Platform**: Custom desktop build
- **CPU**: High-performance x86_64 processor
- **RAM**: 32GB DDR4
- **Swap**: 16GB swapfile (for AI workloads)

### Graphics
- **GPU**: AMD Radeon RX 6800 XT
- **VRAM**: 16GB GDDR6
- **Acceleration**: ROCm stack for AI/ML workloads
- **Monitoring**: radeontop-web for real-time GPU metrics

### Storage
- **Root**: NVMe SSD (OS and applications)
- **Data Disk**: `/mnt/data` - Additional storage mounted at `/dev/disk/by-uuid/60d4fb47-d8fd-4445-adb0-2fd303da775b`
  - Mount options: `defaults,user,rw,exec`
  - Permissions: `755`, owned by `nicholas:users`
  - Backup directory: `/mnt/data/backups/cmos/` (Borg repositories + manual backups)

### Peripherals
- **Webcam**: Logitech C920 HD Pro (configured in `webcam-c920.nix`)
- **Boot Loader**: systemd-boot with EFI

- `47984-47990` - Sunshine game streaming (when enabled)

**Allowed UDP Ports**:
- `137, 138` - NetBIOS
- `5353, 5355` - mDNS/LLMNR

## 🚀 Deployment

### Building Locally
```bash

nixos-rebuild switch --flake .#cmos
```

### Remote Deployment
```bash
# From another host
nixos-rebuild switch --flake .#cmos --target-host cmos.home.arpa
```

```bash
# Test without activating

# Dry run to see changes
```

## 🧪 Troubleshooting

### GPU Issues

**Check ROCm availability**:
```bash
rocminfo
```

**Monitor GPU in real-time**:
```bash
# CLI
radeontop

# Web interface
curl http://localhost:8001
```

**Check Ollama GPU usage**:
```bash
systemctl status ollama
journalctl -u ollama -f
```

### Display Issues

**Check Wayland session**:
```bash
echo $XDG_SESSION_TYPE
```

**Restart display manager**:
```bash
systemctl restart sddm
```

**Check KDE Remote Desktop**:
```bash
systemctl --user status krdp
```

### Storage Issues

**Check data disk mount**:
```bash
df -h /mnt/data
ls -la /mnt/data
```

**Verify backup status**:
```bash
# Check backup timer
systemctl status borgbackup-job-cmos-automated.timer

# Check backup repository
export BORG_PASSPHRASE=$(sudo cat /run/agenix/borg-backup-passphrase)
borg list /mnt/data/backups/cmos/borg
```

**Remount if needed**:
```bash
sudo mount -a
```

### Service Issues

**Check all services**:
```bash
systemctl --failed
```

**Check Ollama**:
```bash
systemctl status ollama
curl http://localhost:11434/api/tags
```

**Check Samba**:
```bash
systemctl status smbd
smbclient -L localhost -U nicholas
```
