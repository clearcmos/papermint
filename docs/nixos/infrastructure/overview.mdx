## 🎯 Overview

This section covers the foundational architecture of our NixOS infrastructure, including multi-host deployment, network topology, and service distribution across the environment.

### 🏠 Host Architecture

```
Internet → Cloudflare → misc.bedrosn.com (Traefik) → Services
                           ↓
    cmos.home.arpa ←→ misc.home.arpa ←→ jimmich.home.arpa
         ↓                 ↓                  ↓
    [Desktop]         [Services]          [Services]
    [AI/GPU]         [Auth/Proxy]        [Media/Files]
    [Gaming]         [Monitoring]        [Backup]
```

### 🌐 Network Architecture

**Internal Network**: `192.168.1.0/24`
- **Gateway**: `192.168.1.1` (Router)
- **DNS**: Cloudflare (1.1.1.1, 1.0.0.1)
- **Local Discovery**: mDNS/Avahi

**Host IP Assignments**:
- **cmos.home.arpa**: `192.168.1.2/24` (static)
- **misc.home.arpa**: `192.168.1.5/24` (static)
- **jimmich.home.arpa**: `192.168.1.10/24` (static)

### 🔌 Network Interfaces

**Per-Host Interface Configuration**:
- **cmos**: `enp8s0` (PCIe network adapter)
- **misc**: `enp3s0f0` (Server-grade dual-port NIC)
- **jimmich**: `enp8s0` (Standard ethernet)

⚠️ **Critical**: Network interfaces are configured **per-host** in `hosts/*/default.nix`. Always verify with `ip link` before configuration changes.

### 🚪 Port Allocation

- **22/tcp** - SSH

**Server Hosts (misc, jimmich)**:

## 🛠️ Deployment Architecture

### 🔄 Flake-Based Deployment

**Structure**:
```
/etc/nixos/
├── flake.nix              # Host definitions and inputs
├── hosts/                 # Host-specific configurations
│   ├── cmos/             # Desktop workstation
│   ├── misc/             # Primary server
│   └── jimmich/          # Media server
├── modules/               # Shared configuration modules
│   ├── core/             # Essential modules (all hosts)
│   ├── desktop/          # GUI modules (cmos)
│   └── services/         # Server modules (misc, jimmich)
└── secrets/              # Encrypted secrets (agenix)
```

### 🔐 Secret Management Architecture

**Agenix Integration**:
- **Public Keys**: Per-host age keys in `keys/`
- **Encrypted Secrets**: In `secrets/` directory
- **Shared Secrets**: Available to multiple hosts
- **Host-Specific**: Secrets for individual hosts

### 🔧 Build Commands

```bash
# Local host rebuild
nixos-rebuild switch --flake .#$(hostname)

# Remote host rebuild
nixos-rebuild switch --flake .#misc --target-host misc.home.arpa

# Test configuration
nixos-rebuild test --flake .#<host>

# Check changes
nixos-rebuild dry-run --flake .#<host>
```

## 🔒 Security Architecture

### 🛡️ Defense in Depth
1. **Network Level**: Cloudflare WAF + DDoS protection
2. **Proxy Level**: Traefik with security headers
3. **Application Level**: Keycloak + OAuth2 Proxy
4. **System Level**: NixOS isolation + systemd hardening

### 🔑 Authentication Flow
```
User → Cloudflare → Traefik → OAuth2 Proxy → Keycloak OIDC → Service
```

### 🏠 LAN vs WAN Access
- **LAN Access**: Direct bypass for `192.168.1.1/32`
- **WAN Access**: Full authentication required
- **Emergency Access**: Documented bypass procedures
