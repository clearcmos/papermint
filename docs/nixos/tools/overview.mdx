## 🎨 Productivity Tools

### 📋 Konsole Utilities

**cpv Command** (Copy Visible):

**Purpose**: Copies all visible text from the currently active Konsole tab to the system clipboard, useful for capturing terminal output without scrollback.

**Usage**:
```bash
# Copy visible text from current tab
cpv

# Works from root shell (auto-detects user session)
sudo -i
cpv
```

**Features**:
- ✅ **Active Tab Detection**: Automatically captures the currently focused Konsole tab
- ✅ **Multi-tab Support**: Works correctly even with multiple Konsole tabs open
- ✅ **Root-aware**: Automatically switches to the user's session when run as root
- ✅ **KDE Integration**: Uses Klipper (KDE clipboard manager) for reliable clipboard management
- ✅ **Wayland Compatible**: Full support for Wayland display server

**How It Works**:
1. Detects the active Konsole window and session via D-Bus
2. Retrieves all displayed (visible) text using Konsole's D-Bus API
3. Copies text to system clipboard via Klipper
4. Persists across terminal sessions

**Use Cases**:
- Capture command output for documentation
- Copy error messages for troubleshooting
- Share terminal sessions with team members
- Quick clipboard access without mouse selection

**Module**: `modules/desktop/konsole-utils.nix`

### ⏱️ CLI Timer with Notifications

**t Command** (timer-cli):

**Purpose**: Simple command-line timer that sends push notifications to your phone via Firebase Cloud Messaging when timers complete. Perfect for timing tasks, pomodoro sessions, cooking, or any timed activity.

**Usage**:
```bash
# Basic timer formats
t 60s          # 60 seconds
t 3m           # 3 minutes
t 2h           # 2 hours

# With spaces (flexible parsing)
t 60 seconds   # 60 seconds
t 3 minutes    # 3 minutes
t 2 hours      # 2 hours

# Decimal times
t 1.5h         # 1 hour 30 minutes
t 2.25m        # 2 minutes 15 seconds
t 30.5s        # 30.5 seconds
```

**Features**:
- ✅ **Flexible Time Parsing**: Supports s/sec/seconds, m/min/minutes, h/hr/hours with decimals
- ✅ **Push Notifications**: FCM notifications to your phone when timer completes
- ✅ **Clear Feedback**: Shows timer duration and completion time on start
- ✅ **Secure Secrets**: Uses agenix for Firebase credentials (service account + device token)
- ✅ **OAuth2 Authentication**: Secure FCM HTTP v1 API with JWT-based authentication

**Example Session**:
```bash
$ t 25m
⏱️  Timer set for 25m
🕐 Will notify at 03:45:30 PM

# ... 25 minutes later ...

⏰ Timer complete! Sending notification...
✅ Notification sent!
```

**How It Works**:
1. Parses time argument into seconds (supports flexible formats)
2. Calculates end time and displays countdown info
3. Sleeps for the specified duration (can Ctrl+C to cancel)
4. Generates OAuth2 access token using Firebase service account JWT
5. Sends data-only FCM notification to registered device token
6. Displays success/failure status

**Push Notification Format**:
- **Title**: "⏰ Timer Complete"
- **Body**: "{duration} timer has finished" (e.g., "25m timer has finished")
- **Priority**: High (Android) for full-screen alarm-style alerts

**Configuration**:
- **Module**: `modules/desktop/timer-cli.nix`
- **Secrets** (agenix):
  - `firebase-service-account.age` - FCM service account credentials
  - `fcm-device-token.age` - Target device FCM token
- **Dependencies**: Python 3 with requests, pyjwt, cryptography

**Use Cases**:
- 🍅 **Pomodoro Sessions**: `t 25m` for focused work intervals
- ⏲️ **Cooking Timers**: `t 45m` for recipes or baking
- 💪 **Workout Intervals**: `t 90s` for exercise circuits
- 🧘 **Meditation**: `t 10m` for timed meditation sessions
- 📚 **Study Sessions**: `t 1.5h` for deep work blocks

**Keyboard Interrupt**:
```bash
$ t 1h
⏱️  Timer set for 1h
🕐 Will notify at 04:30:00 PM
^C
⚠️  Timer cancelled
```

**Module**: `modules/desktop/timer-cli.nix`
**Full Documentation**: [timer-cli.md](timer-cli.md)
