## The Problem

Google Workspace administration at scale (1,000+ users) involves complex operations across Drive, Gmail, Calendar, Groups, and Admin SDK. Many operations — offboarding file transfers, shared drive ACL audits, Gmail Vault restorations, calendar management — require multi-step API workflows that the admin console doesn't support.

## What Was Built

28 purpose-built Google API automation tools (~15,900 lines of Python) plus GAMADV-XTD3 scripts and Apps Script integrations.

### Notable Technical Details

**Vault Matter Restoration (917 lines):**
- Creates temporary Vault Matter for export (automatically cleaned up)
- Rebuilds conversation threading using In-Reply-To and References headers
- Topologically sorts messages so parent messages are inserted before replies
- Uses Gmail `messages.insert()` API (100 msg/min rate limit management)
- Checkpoint/resume system for large restorations
- Progress tracking with estimated completion time

**Archive Cleanup (1,294 lines):**
- Combines Gmail label audit + Drive folder analysis in one workflow
- Queries BigQuery for file last-access data
- Concurrent processing with ThreadPoolExecutor
- Writes results directly to dated Google Sheets tabs
- Generates outreach emails to file owners about inactive content

## Impact

- Offboarding file transfers that took hours of manual work are now automated with full audit trails
- Vault matter restoration preserves conversation threading — previously, restored emails lost their thread context
- Shared drive ACL audits that were impossible manually (thousands of drives, hundreds of thousands of files) now run on schedule
- Security audits (public calendar sharing, "anyone with link" files, orphaned shared drives) catch exposure that was previously invisible
- Archive cleanup with BigQuery access data enables data-driven decisions about content retention
- The entire suite reduces Google Workspace administration from a full-time activity to automated workflows with human oversight
